# Track e-learning — New Hub（プロトタイプ）

Next.js 製の e ラーニング統合 UI プロトタイプ（管理画面 + 受講者 Skill Hub / Track）。

## クイックスタート

```bash
npm install
npm run dev
```

ブラウザで `http://127.0.0.1:3000`（推奨）または `http://localhost:3000`。
